## 🔧 Configuração e Desenvolvimento

### Pré-requisitos

- [Bun](https://bun.sh/) instalado

### Instalação

```sh
bun install
```

### Desenvolvimento Local

```sh
bun dev
```

### Build para Produção

```sh
bun run build
```

### Linting e Formatação

```sh
bun lint
bun format
```
